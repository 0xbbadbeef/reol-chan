# REOL-CHAN


## Requirements
- Postgres (For storing media)
- NodeJS > 7.0

## To initialize:
Make sure you have the latest NodeJS version.
```
npm install
```
After this, create a `.env` file at the root directory, using `.env.sample` as a sample.
